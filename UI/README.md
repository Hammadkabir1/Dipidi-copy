# Jenee Realtime Voice System

## Setup

1. Install websockets:
```bash
pip install websockets
```
2. cd UI 

3. In a separate terminal, start Streamlit:
```bash
python -m streamlit run app.py
```


## Usage

1. Go to Voice tab
2. Click "Start Call"
3. Allow microphone access
4. Speak naturally with Jenee
5. Click "End Call" when done
